
This project is a very basic portfolio website telling you a little bit about me and my experience in programming.

To run this code you can either use this link to the deployed website https://lordbaldwin1.github.io/ or you can
download the files and run them yourself by opening index.html.

For this website I used https://www.freecodecamp.org/news/how-to-build-a-developer-portfolio-website/ as an outline.

https://developers.google.com/fonts/docs/getting_started was used to stup fonts.

https://www.w3docs.com/snippets/javascript/how-to-scroll-to-the-top-of-the-page-using-javascript.html#:~:text=To%20make%20the%20page%20jump,of%20coordinates%20in%20the%20document. was used to fix
my scroll to top of page button.

https://www.sitepoint.com/create-perfect-hero-image-portfolio-site/ was used to understand what hero images
on a portfolio are.

https://www.w3schools.com/howto/howto_css_hero_image.asp was used to edit hero image.

https://formspree.io/ is used to save data entered in the form.
